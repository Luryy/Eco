## :clipboard: Usage:

### Backend

``` 
cd server
npm install
npm install --production
npm run knex:migrations
npm run knex:seeds
```

### Frontend

``` 
cd web
npm install
npm install --production
npm start
```

### Mobile

``` 
cd mobile
npm install
npm install --production
npm start
```
